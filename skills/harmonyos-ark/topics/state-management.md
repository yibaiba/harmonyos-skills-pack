# 状态管理主题

## Scope
- 单页面状态、跨组件状态共享、数据绑定与一致性管理

## Key Concepts
- 单向数据流
- 局部状态
- 全局状态
- 状态同步
- 数据绑定
- 不可变更新

## Official Entrypoints
- HarmonyOS 状态管理与数据绑定文档
- ArkUI 状态装饰器官方说明
- 状态管理相关示例工程

## Quick Q&A
- 小型页面和复杂页面分别推荐哪种状态策略
- 跨组件共享状态如何避免耦合
- 数据绑定性能问题通常如何排查

## Common Pitfalls
- 把临时状态和全局状态混用
- 状态更新路径不统一导致竞态问题
- 大对象频繁更新造成界面卡顿

## Boundaries
- 不覆盖后端分布式状态一致性协议
- 不覆盖数据库事务理论细节
