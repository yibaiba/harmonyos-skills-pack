## 使用方式

- 声明式 UI 语法/状态管理 → `arkui-core.md`
- 组件开发（布局/列表/文本/弹窗） → `arkui-components.md`
- 动画/手势/交互 → `arkui-interaction.md`
- 主题/国际化/无障碍/性能 → `arkui-advanced.md`
- HDS 设计套件组件 → `ui-design-kit.md`
